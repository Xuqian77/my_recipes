# Qian's Chef's Table - 私家菜谱点餐系统

这是一个专为私家大厨设计的菜谱管理与客座点餐 Web 应用。它不仅能帮助您记录和整理自己的拿手好菜，还能在亲朋好友来访时，作为一个精美的电子菜单供他们点餐。

## ✨ 主要功能

- **精美菜单展示**：采用优雅的“暖色有机”设计风格，为客人提供如高级餐厅般的点餐体验。
- **全方位菜谱记录**：
  - **基础信息**：菜名、口味标签（如甜辣、鲜香）。
  - **烹饪参数**：制作时长、难易程度（星级打分）。
  - **详细内容**：完整的食材清单、文字介绍的完成过程。
- **本地图片存储**：支持直接上传菜品完成照片，以二进制形式安全存储在本地数据库中，无需外部链接。
- **视频教学链接**：可附带视频链接，方便随时回顾烹饪细节，防止遗忘。
- **管理员模式**：支持菜谱的增加、修改和删除，轻松管理您的私人小厨房。
